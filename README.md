# nnigam96.github.io

This is the source code for my personal website, hosted at [nnigam96.github.io](https://nnigam96.github.io). The site serves as a portfolio and digital resume, designed to showcase my professional experience, technical skills, and personal interests.

## About the Site

The website is built using [Astro](https://astro.build), a modern static site generator that enables fast, content-focused websites. Content is managed through Markdown files, making it easy to maintain and update project case studies.

Key highlights:
- Built with Astro for optimal performance
- Markdown-based content management for projects
- TypeScript with strict mode for type safety
- Modern design system with CSS variables
- Responsive and accessible design

## Technologies Used

- [Astro](https://astro.build) - Static site generator
- TypeScript - Type-safe development
- Markdown - Content management
- Zod - Schema validation for content collections

## Project Structure

```
src/
  components/     # Reusable Astro components
  content/        # Markdown content (projects)
  layouts/        # Page layouts
  pages/          # Route pages
  styles/         # Global CSS
```

## Development

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Deployment

The site is automatically deployed using GitHub Pages from the `main` branch. The build output is in the `dist/` directory.
